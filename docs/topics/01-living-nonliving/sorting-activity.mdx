# Sorting Living and Non-living Things

Now you know about living and non-living things! Let's practice telling them apart!

## How to Sort

Follow these steps to tell if something is living or non-living:

### Step 1: Ask "Can it grow?"
- If YES ‚Üí Might be living!
- If NO ‚Üí Might be non-living!

### Step 2: Ask "Does it need food?"
- If YES ‚Üí Might be living!
- If NO ‚Üí Might be non-living!

### Step 3: Ask "Can it move by itself?"
- If YES ‚Üí Might be living!
- If NO ‚Üí Might be non-living!

### Step 4: Ask "Can it have babies?"
- If YES ‚Üí It's living! ‚úì
- If NO ‚Üí It's non-living! ‚úì

:::tip Quick Check
If it can do ALL 5 signs of life (grow, eat, breathe, move, have babies), it's **LIVING**!
If it cannot do these things, it's **NON-LIVING**!
:::

## Let's Sort Together!

### Example 1: A Cat üê±

- Can it grow? **YES** (Kittens grow into cats)
- Does it need food? **YES** (Cats eat food)
- Can it breathe? **YES** (Cats breathe air)
- Can it move by itself? **YES** (Cats walk and run)
- Can it have babies? **YES** (Cats have kittens)

**Answer: A cat is LIVING!** ‚úì

### Example 2: A Chair ü™ë

- Can it grow? **NO** (Chairs stay the same size)
- Does it need food? **NO** (Chairs don't eat)
- Can it breathe? **NO** (Chairs don't need air)
- Can it move by itself? **NO** (You have to move it)
- Can it have babies? **NO** (Chairs can't make baby chairs)

**Answer: A chair is NON-LIVING!** ‚úì

### Example 3: A Tree üå≥

- Can it grow? **YES** (Trees start small and grow big)
- Does it need food? **YES** (Trees make food from sunlight)
- Can it breathe? **YES** (Trees take in air through leaves)
- Can it move? **YES** (Trees grow toward sunlight)
- Can it have babies? **YES** (Trees make seeds)

**Answer: A tree is LIVING!** ‚úì

## Tricky Questions!

### "Is the sun alive?"

The sun is bright and gives us light, but:
- It doesn't eat food ‚ùå
- It doesn't have babies ‚ùå
- It's not a living thing!

**The sun is NON-LIVING, but it helps living things!**

### "Are seeds alive?"

Yes! Seeds are living things because:
- They can grow into plants ‚úì
- They need water ‚úì
- They are baby plants waiting to grow ‚úì

**Seeds are LIVING!**
